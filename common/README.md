# common

Shared utility code used across the repository.

## Main files

- `io_utils.py`  
  JSON and CSV helpers.

- `sampling.py`  
  Deterministic subset and stratified-subset helpers.

- `__init__.py`  
  Re-exports the main shared helpers.

## Notes

It provides reusable I/O and sampling utilities for the synthetic and ChaosNLI pipelines.
